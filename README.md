# Portafolio-Moviles
ITIID - 4 ANGEL DAVID SERRA PERALES  DESARROLLO DE APLICACIONES MOVILES


01_holaMundo.dart

Este ejercicio sirve para conocer la estructura mínima de un programa en Dart. Permite entender cómo se ejecuta el método principal (main) y cómo mostrar información en consola mediante la salida estándar.

02_DataTypes.dart

Introduce los tipos de datos fundamentales del lenguaje, mostrando cómo declarar variables y almacenar distintos tipos de información como números, texto y valores booleanos. Su propósito es comprender cómo Dart maneja los datos en memoria.

03_finalConst.dart

Explica el uso de variables inmutables. Permite diferenciar entre valores que pueden cambiar durante la ejecución y aquellos que deben mantenerse constantes, ayudando a mejorar la seguridad y organización del código.

04_comments.dart

Muestra cómo documentar el código mediante comentarios. Su finalidad es facilitar la lectura y mantenimiento del programa sin afectar la ejecución.

05_mathOperators.dart

Se enfoca en el uso de operadores matemáticos para realizar cálculos básicos. Permite comprender cómo se ejecutan operaciones aritméticas dentro de un programa.

06_operators.dart

Amplía el uso de operadores incluyendo comparaciones y operaciones lógicas. Su objetivo es entender cómo evaluar condiciones y obtener resultados verdaderos o falsos.

07_read.dart

Introduce la lectura de datos desde el teclado. Permite que el usuario interactúe con el programa ingresando información durante la ejecución.

08_helloName.dart

Aplica la entrada de datos para personalizar la salida del programa. Su propósito es mostrar cómo utilizar información ingresada por el usuario dentro de mensajes dinámicos.

09_sum.dart

Ejemplo práctico de procesamiento de datos ingresados por el usuario. Enseña cómo convertir texto en números y realizar operaciones con ellos.

10_concat.dart

Demuestra la unión de cadenas de texto. Permite entender cómo combinar información para generar mensajes más completos.

11_mayor.dart

Introduce la toma de decisiones mediante estructuras condicionales simples. El programa evalúa valores y determina cuál cumple una condición específica.

12_nestedIF.dart

Explica el uso de condiciones anidadas. Sirve para resolver problemas donde se requiere evaluar múltiples decisiones en orden lógico.

13_area.dart

Aplica fórmulas matemáticas dentro de un programa. Su finalidad es resolver problemas prácticos mediante cálculos basados en datos ingresados.

14_tablas.dart

Muestra el uso de ciclos repetitivos para automatizar tareas. Permite generar resultados múltiples sin repetir código manualmente.

15_grades.dart

Ejercicio orientado a clasificar resultados según rangos. Enseña cómo estructurar decisiones cuando existen varios posibles resultados.

16_suma.dart

Refuerza el uso de ciclos y acumuladores para realizar sumatorias. Ayuda a comprender cómo almacenar resultados parciales durante una repetición.

17_labes.dart

Introduce el uso de etiquetas en estructuras de control para manejar mejor el flujo del programa, especialmente en ciclos anidados.

18a_functions.dart

Presenta la creación de funciones para organizar el código. Permite dividir un programa en bloques reutilizables que cumplen tareas específicas.

18b_functions.dart

Amplía el uso de funciones incorporando parámetros y valores de retorno. Su objetivo es mejorar la reutilización y modularidad del programa.

19_cousins.dart

Ejercicio aplicado que integra lectura de datos, decisiones y lógica condicional para resolver un problema más completo.

profesor_04.dart

Ejemplo adicional enfocado en reforzar conceptos básicos vistos previamente, mostrando una forma alternativa o guiada de resolver un problema.

profesor_05.dart

Ejercicio complementario que consolida el uso conjunto de variables, operadores y estructuras de control en un solo programa.
